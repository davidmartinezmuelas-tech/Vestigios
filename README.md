# Vestigios

Dark fantasy RPG in active development, built with Godot 4.

## Overview

Vestigios is a narrative RPG set in a world where two kingdoms, Velthar and Kethara, have been trapped in a long war. Behind the political conflict is a deeper cosmic threat: ancient entities, hidden influence, and characters forced to discover the real nature of the world they inhabit.

The project is currently focused on game design, narrative architecture, core systems, and technical foundations before moving into full implementation.

## Current Status

Design phase complete. Implementation starting.

- Worldbuilding and main regions defined
- Playable character roster designed
- Core lore and NPC documentation written
- Gameplay systems planned
- Godot 4 architecture direction defined

## Core Systems

- Champion system tied to ancient cosmic entities
- Corruption zones with risk/reward exploration
- Branching dialogue with skill checks
- Tactical combat with positioning and character abilities
- Character progression influenced by narrative events
